# QUIZ-TEST-GENERATOR

Tool that will enable one to randomly generate a quiz test questionnaire containing the specified number of questions from the question bank. The question bank
and other related data are persisted in files. The system ensures that the file exits and the path is specified or else a
file is created with the specified name and stored in default location. Options are provided for separate question bank for separate subjects. User has the option of selecting the subject for which the question bank is to be made or the questionnaire is to be generated.
